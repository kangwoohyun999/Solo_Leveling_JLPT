# Solo_Leveling_JLPT

solo_leveling_word 라는 웹사이트를 만들건데 java, python, c++ 등 가장 효율적인 언어로 작성해줘.

모든 화면은 스크롤 가능하게.

로그인 화면 거쳐서 메인 화면으로 들어가게 만들거야. vercel 플랫폼으로 구동할거야.

메인 화면 : 
[상단] 왼쪽 : 로고, 중간 : 앱 이름, 오른쪽 : 목록(가로 세줄로 표기)
[중간 및 하단] 6개로 열 형태의 버튼으로 구성, 위에서부터 이름 : 연계 (나중에 디테일 추가할 것. 버튼만 만들어두는 용), N5, N4, N3, N2, N1 으로 구성.

목록 버튼 클릭 시:
계정 정보(정보 수정), 오답노트

N1~N5 버튼 클릭 시 :
[상단] 해당하는 이름 표시 ex) N1
[중간 및 하단] 여러개의 열 형태의 박스로 구성. 박스 하나 당 {왼쪽 : 일본어 단어, 중간 : 히라가나 발음, 오른쪽 : 한국어로 뜻 표시} 이렇게 구성해줘.
하단 오른쪽 작은 원형으로 Q가 써져있는 버튼. (스크롤해도 내려가지 않게 고정)

Q 버튼 클릭 시 :
문제 풀 개수 정하는 화면. 문제수 정하면 퀴즈 화면으로 이동.

퀴즈 화면 :
[상단(화면 중간까지)] 박스로 구성. 일본어 단어 중앙에 크게 배치. 단어 아래에 작게 발음 제공.
[하단] 3열로 구성된 버튼. 한개만 맞는 뜻, 나머지는 랜덤 뜻으로.

jword.csv 파일 나중에 추가할건데 포함해서 코드 작성해줘 나중에 다시 수정할거야.
